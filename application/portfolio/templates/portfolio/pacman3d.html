Description

Le but de ce projet était de developper un pacman en 3D à l'aide d'openGL 3 et de la SDL 1.2, en C++ 11.

Il utilise les bibliothèques suivantes en plus d'openGL et la SDL:

  • Assimp : Pour l'import des modèles 3D
  • Freetype : Pour le texte affiché en jeu
  • JSON for C++ : Pour gérer le format json facilement

Le projet utilise également python pour la génération du niveau au format json.

Equipe de 3 développeurs

Un executable est disponible sur le lien suivant : Pacman3D Windows .
