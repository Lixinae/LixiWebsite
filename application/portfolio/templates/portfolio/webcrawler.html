Description

Ce crawler simple permet de scraper des sites web pour récupérer des données, il se base sur la bibliothèque Beautifoul Soup 4

Il a tout d'abord été développé en tant que script simple pour récupérer les pdf des mes professeurs à la fac

Il a évolué pour être capable de récuperer tout type de fichier sur un site, tout en gardant l'architecture du site

Il n'est pas possible de se log sur un site et d'y récupérer le contenu

Il fonctionne uniquement avec les liens direct vers les ressources
