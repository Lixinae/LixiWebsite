Description

Le but du projet est de developper un générateur de texte à partir d'un acronyme.

On peut choisir la langue dans laquelle on veut le résultat.

Le texte est aléatoire, donc générer un texte à partir d'un acronyme 2 fois de suite ne donnera jamais le même résultat.
